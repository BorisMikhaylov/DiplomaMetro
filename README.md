# Диплом: перераспределение маршрутов при закрытии станции метро

**Тема:** «Разработка модели перераспределения маршрутов и расписаний в случае закрытия станций московского метро»

**Станция-кейс:** Тропарево, Сокольническая линия

---

## Структура репозитория

```
DiplomaMetro/
├── analysis/
│   ├── 02_get_coordinates.py       # координаты станций (Overpass/Nominatim)
│   ├── 03_hourly_profiles.py       # почасовой профиль по всем станциям
│   ├── 04_troparyovo_profile.py    # профиль Тропарево
│   ├── 05_simulate_closure.py      # базовая симуляция закрытия
│   ├── 06_ngpt_near_troparyovo.py  # маршруты НГПТ у станции
│   └── 07_route_redistribution.py  # модель перераспределения маршрутов
├── reports/
│   ├── data_audit.md               # аудит данных НБС
│   └── progress_coordinates_and_hourly.md
├── setup_data.sh                   # загрузка данных из приватного репо
└── CLAUDE.md                       # описание проекта для AI-ассистента
```

## Данные

Данные хранятся в приватном репозитории [DiplomaData](https://github.com/BorisMikhaylov/DiplomaData).

```bash
bash setup_data.sh
```

**PASS_ALL_202503242210.csv** (~12 ГБ, 89 млн строк) — выгрузка НБС за 10–16 марта 2025.
Файл слишком большой для git. Запросить у автора.

## Запуск

```bash
# 1. Загрузить данные
bash setup_data.sh

# 2. Профиль пассажиропотока Тропарево
python3 analysis/04_troparyovo_profile.py

# 3. Маршруты НГПТ у станции
python3 analysis/06_ngpt_near_troparyovo.py

# 4. Модель перераспределения
python3 analysis/07_route_redistribution.py
```

## Результаты (Тропарево)

- Средняя нагрузка в будни: **~20 700 чел/день**
- Пик: **08:00** (~2 630 чел/час)
- Маршруты НГПТ у станции: **229, 283, т6, 645, 111, 191** + ~6 неидентифицированных
- Без усиления НГПТ покрывает **~23%** пикового спроса
- После усиления (интервал 4 мин): **~50%**
- Остаток закрывается **шаттлами** (3 автобуса в час пик)

## Требования

- Python 3.9+
- Стандартная библиотека (csv, collections, datetime) — pandas не требуется
