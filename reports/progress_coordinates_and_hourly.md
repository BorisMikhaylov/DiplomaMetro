## Следующие шаги для модели закрытия Тропарево

### Приоритет 1 — Граф ближайших станций
Ближайшие по Сокольнической линии:
- **Юго-западная** (сев., +1 остановка) — основной маршрут объезда для едущих в центр
- **Румянцево** (юж., +1 остановка) — объезд для едущих на юг/Саларьево

### Приоритет 2 — НГПТ в радиусе 500м от Тропарево
Нужно найти автобусные маршруты рядом со ст. Тропарево (координаты ~55.6222, 37.4094).
Источники: GTFS Московского транспорта / data.mos.ru / ручной поиск по картам.

### Приоритет 3 — Функция `simulate_closure('Тропарево')`
На основе готового профиля:
1. Взять спрос по часам из `troparyovo_hourly.csv`
2. Распределить по соседним станциям (Юго-западная + Румянцево + НГПТ)
3. Оценить перегруз с учётом вместимости (нужны данные о провозной способности)
4. Вывести дефицит и рекомендации
